# Basic Chatbot

A simple rule-based chatbot developed using Python.

## Features
- Greets the user
- Responds to predefined messages
- Uses if-elif statements
- Runs in a loop until the user types `bye`

## Technologies Used
- Python 3

## Concepts Used
- Functions
- While Loop
- If-Elif-Else
- Input/Output

## How to Run

```bash
python basic_chatbot.py
```
